## 💻 EXECUTION PROTOCOLS
To run any module locally, navigate to the specific directory and execute the following:
*   **Web Modules (01-06)**: Open `index.html` in any modern browser.
*   **System Modules (08-09)**: Execute the `.sh` (Linux) or `.ps1` (Windows) scripts in your terminal.
*   **Logic Modules (07, 10)**: Use `tsc` (TypeScript) or `git` commands as detailed in their specific directories.
